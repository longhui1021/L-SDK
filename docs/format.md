# 格式转化

## 1. digitUppercase(num)

> 数字转化为大写金额

- 参数类型 `Number`
- 返回值类型 `String`

```javascript
import { digitUppercase } from 'l-sdk'

console.log(digitUppercase(935094521)); // 玖亿叁仟伍佰零玖万肆仟伍佰贰拾壹元整

```

## 2. intToChinese(num)

> 数字转化为中文数字

- 参数类型 `Number`
- 返回值类型 `String`

```javascript
import { intToChinese } from 'l-sdk'

console.log(intToChinese(935094521)); // 九亿三千五百零九万四千五百二十一

```
